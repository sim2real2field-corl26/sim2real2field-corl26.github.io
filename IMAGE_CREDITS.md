# Headshot image provenance

Each headshot in `images/` was sourced from the person's own official/personal page
(or a reputable institutional/official source where their own page had no photo), then
downscaled to <=480px and stripped of metadata for web use. Identity was verified by
matching the affiliation stated in the proposal. **Please confirm each photo is the
correct person and that you're comfortable using it before treating the site as final.**
