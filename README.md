# Sim2Real2Field — CoRL 2026 Workshop companion site

A single-page static site (no build step). Companion to the workshop proposal
*Sim-to-Real-to-Field (SRF): The Science of Transfer*.

## Files
- `index.html` — the page (all content lives here)
- `styles.css` — styles (navy `#0b3a6f` + gold `#f2b50a`, Palatino/Tufte feel)

## Preview locally
Open `index.html` in any browser (keep `styles.css` in the same folder).

## Deploy to GitHub Pages
This repo is named **`sim2real2field-corl26.github.io`** under the org
**`sim2real2field-corl26`**, so GitHub Pages serves it at the root URL:
`https://sim2real2field-corl26.github.io/`.

1. Push `index.html` and `styles.css` to the default branch (repo root).
2. Settings → Pages → Source: deploy from a branch, default branch / root.
